# График кривой r = |φ| и вычисление площади методом Монте-Карло

Этот репозиторий содержит программу на C++ с использованием библиотеки Qt, которая строит график кривой `r = |φ|` в полярной системе координат и вычисляет площадь области, ограниченной этой кривой, с использованием метода Монте-Карло.

## Описание проекта

### Основные функции:
1. **Построение графика кривой `r = |φ|`**:
   - Программа строит график кривой `r = |φ|` для углов `φ ∈ [-π, π]`.
   - График отображается в декартовой системе координат, преобразованной из полярной.

2. **Вычисление точной площади**:
   - Точная площадь области, ограниченной кривой, вычисляется аналитически по формуле:
     ```
     S_точная = (π^2) / 2
     ```

3. **Вычисление площади методом Монте-Карло**:
   - Метод Монте-Карло используется для приближенного вычисления площади.
   - Программа генерирует случайные точки в прямоугольной области и подсчитывает количество точек, попавших внутрь области, ограниченной кривой.
   - Приближенная площадь вычисляется по формуле:
     ```
     S_Монте-Карло = (Количество точек внутри / Общее количество точек) * S_прямоугольника
     ```

4. **Оценка точности метода**:
   - Программа вычисляет относительную погрешность метода Монте-Карло по сравнению с точным значением площади.

## Структура репозитория

- **`mainwindow.cpp`**:
  - Содержит код для создания графического интерфейса и построения графика кривой.
  - Реализует метод Монте-Карло для вычисления площади.

- **`SquareMC.cpp`**:
  - Содержит реализацию методов для вычисления точной площади, площади методом Монте-Карло и оценки точности.
